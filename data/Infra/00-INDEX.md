# Infra 문서 인덱스

이 문서는 `C:\dev\Infra`에서 가장 먼저 보는 시작점이다.

## 빠른 시작

1. 포맷 직후: `setup.md`
2. 비밀키 설정: `secrets-setup.md`
3. 포맷 후 점검: `checklist-post-format.md`
4. 변경 기록 확인: `logs/2026-02.md`
5. 새 문서 작성: `runbooks/template.md`

## 문서 맵

- `setup.md`: Windows + WSL + Ubuntu 온보딩 절차
- `secrets-setup.md`: secrets 저장 위치, 로딩 방식, 점검 명령
- `checklist-post-format.md`: 포맷 후 필수 체크리스트
- `logs/2026-02.md`: 월간 변경 로그
- `runbooks/template.md`: 표준 런북 템플릿

## 원칙

- 실제 토큰/API 키 값은 문서에 기록하지 않는다.
- 실행한 내용은 월간 로그에 남긴다.
- 새 설정을 추가하면 체크리스트/런북을 같이 갱신한다.
